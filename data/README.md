Coloca aqui los ficheros CSV brutos descargados desde Drive.

Ejemplo:
- `data/raw/1RFEF_2025-26.csv`
- `data/raw/2RFEF_2025-26.csv`

Estos archivos pueden mantenerse fuera de Git si contienen datos que no quieras versionar.

Configuracion editable del radar:
- `data/config/radar_metrics.csv`

Ese fichero si debe versionarse, porque define que metricas usa la app para cada grupo de posicion.
